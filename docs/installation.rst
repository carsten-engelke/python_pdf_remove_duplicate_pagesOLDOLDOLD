============
Installation
============


Stable release
--------------

To install Python PDF Remove Duplicate Pages, run this command in your terminal:

.. code-block:: console

    $ pip install python_pdf_remove_duplicate_pages

This is the preferred method to install Python PDF Remove Duplicate Pages, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Python PDF Remove Duplicate Pages can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/carsten-engelke/python_pdf_remove_duplicate_pages

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/carsten-engelke/python_pdf_remove_duplicate_pages/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/carsten-engelke/python_pdf_remove_duplicate_pages
.. _tarball: https://github.com/carsten-engelke/python_pdf_remove_duplicate_pages/tarball/master
